  • cp -r dir1 dir2 //复制，将dir1的所有内容复制到dir2，如果dir2不存在的话自动创建dir2

  • mv /usr/lib/* /zone //将/usr/lib/下所有的内容移动到zone下
  • rz -be //解决上传压缩包乱码
